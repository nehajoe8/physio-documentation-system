AI-Powered Physiotherapy Documentation System

HOW TO RUN THE PROJECT

1. Open File Explorer.
2. Go to this folder:
   C:\Users\nehaj\Documents\Codex\2026-06-02\as-part-of-the-training-cell\work\physio-doc-system
3. Double-click index.html.
4. The project will open in your browser.
5. Fill the form.
6. Click Generate SOAP Note.
7. Click Save Session if you want to save it in the browser.
8. Click Download Report to download the generated physiotherapy report.

WHERE TO ENTER OR EDIT CODE

1. Right-click index.html.
2. Choose Open with Notepad or Visual Studio Code.
3. Edit the code.
4. Press Ctrl + S to save.
5. Refresh the browser page to see changes.

WHAT THIS FIRST VERSION DOES

1. Takes patient and physiotherapy assessment details.
2. Generates a structured SOAP note.
3. Suggests a basic home exercise plan.
4. Saves sessions in the browser using local storage.
5. Downloads the report as a text file.

IMPORTANT

Use dummy or anonymized patient data only. Do not enter real patient names or private medical information in this academic prototype.
